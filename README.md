Spring platform for [WebApp Application](https://hmis.netlify.app/)

Steps to install
---
- Clone this repository
- Run `gradle clean` 
- Then run `gradle bootRun`
- Read the Api documentations from http://localhost:8080/swagger-ui.html#/
